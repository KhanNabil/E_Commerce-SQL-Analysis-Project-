# E_Commerce-SQL-Analysis-Project-

## Project Overview

This project simulates an e-commerce database and uses SQL to answer real business questions.

The database contains:

* Customers
* Orders
* Order Details
* Products

The goal is to generate business insights using SQL queries rather than simply retrieving data.

## Skills Demonstrated

* Joins
* Group By
* Having
* Aggregate Functions
* Scalar Functions
* Date Functions
* Subqueries
* Views
* CASE Statements
* Business Reporting

## Business Questions Solved

### Customer Analysis

* Total spending by customer
* Customers spending above average
* VIP, Regular and Basic customer categorization
* Confirmed revenue by customer

### Product Analysis

* Most sold products
* Products selling above average quantity
* Product revenue analysis
* Product performance categorization

### Order Analysis

* Confirmed, Pending and Cancelled orders
* Revenue by order status
* Order age analysis
* Monthly order reporting

## Key Insights

* Identified top-performing customers.
* Categorized customers based on spending behavior.
* Identified products performing above average.
* Measured revenue contribution by order status.
* Created executive-style business reports using SQL.

## Technologies Used

* Microsoft SQL Server
* SQL Server Management Studio (SSMS)

## Conclusion

This project demonstrates practical SQL skills used in data analysis and business reporting. The focus was on solving business problems using SQL rather than only performing database operations.
