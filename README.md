# 🛒 E-Commerce Database Management System

##  Project Overview

This is my second SQL portfolio project, built using Microsoft SQL Server. The objective of this project is to simulate a real-world e-commerce database while applying SQL concepts to solve business problems rather than simply demonstrating SQL syntax.

The database is designed to manage customers, products, orders, order details, and payments. Using this data, I created business-oriented reports that provide insights into customer behavior, product performance, revenue generation, inventory management, payment analysis, and executive decision-making.

#  Database Schema

The database consists of five relational tables:

* Customer
* Orders
* OrderDetails
* Product
* Payments

The tables are connected using Primary Keys and Foreign Keys to maintain referential integrity and simulate a real e-commerce database.
